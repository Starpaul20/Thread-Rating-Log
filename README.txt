## INFORMATION ##

Thread Rating Log (1.1.1) for MyBB 1.6
Created by: Starpaul20
Copyright: ©2010
License: GPL

Adds a log of all thread ratings viewable in the Admin CP. The log will tell you who rated what, and the rater's IP address.

The log can be found alongside of the other board logs under the 'Tools & Maintenance' section.

This plugin offers full language support.


## INSTALLATION ##

1. Upload all files above, keeping the file structure intact.
2. Go to Configuration > Plugins
3. Click "Activate"
4. Enjoy!


## UPDATING ##

If you're updating from any previous version, you only need to overwrite old files with the new ones, no need to deactivate.
